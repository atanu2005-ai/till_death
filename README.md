# Till death
A simple terminal-based battle game written in Java where players choose characters and fight until one survives.
## Features
- Character selection
- Random toss system to decide who starts
- Turn-based attacks
- Different attack mode with limitations
- Health tracking system
- Simple terminal UI
## How to Run
1. Clone the repository

2. Compile the Java file

```bash
javac till_death.java
```

3. Run the code
```bash
java till_death
```
